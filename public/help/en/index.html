Online-Help for Kiwi

Background

Latin characters in Unicode is a standard created by the Coordination Office for IT Standards (KOSIT) on behalf of the IT Planning Council (for the Federal Republic of Germany). In summary, the standard defines what Unicode-compliant processing for the German public administration is. With the decision 2014/04, the IT Planning Council at its 13th meeting defined the characterset Latin characters in UNICODE, which must be supported by IT procedures in the Federal Republic of Germany.

In addition to the correct processing of Latin characters in Unicode, it is also important to support users in entering the required characters as easily and conveniently as possible. For this purpose, the tool Kiwi (Kinderleicht Wortbestandteile integrieren; engl. Integrate word components with ease) was created, which can be used without installation on all clients (PC, Smartphone, Tablet, etc.).

Concept / Idea

The idea of the Kiwi application is not to replace the input device, but to make the characters accessible and findable that are not easy to find on it. Every device already has an input option perfectly adapted to its typical use: keyboard, on-screen keyboard, etc.
Characters that cannot be entered directly via the standard input device (in the example below the "š") can be delimited via the optically related character (base character; here "s") and then be found in the reduced selection simply by optical comparison.
The integration with the application is done by means of the system clipboard, like with the usual tools like desktop calculator.

Quickstart

The name "Miloš" is to be typed with a German keyboard on which the character "š" cannot be found:

  • In Kiwi, "Milos" is entered via the keyboard as usual. Here you first use the German "s" (only the base character, without the check mark above it).
  • Use the enter key (↵) or the magnifying glass icon to start a search.
  • The "keyboard" now shows different "s" variants. With a mouse click, the searched "š" is displayed in the search result selected and automatically transferred to the input field (the simple "s" is replaced!).
  • Ctrl+C (or the symbol with the two overlapping rectangles) will copy all the characters in the input field to the system clipboard.
  • With Ctrl+V (or the usual methods for pasting from the clipboard) the name is entered in the form field of the application.
